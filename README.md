# Sunu-Idées

## Description

Sunu-Idées est une application web collaborative développée en HTML, CSS et JavaScript.

Elle permet aux apprenants de la Promo P9 de proposer, consulter, modifier et supprimer des idées de manière anonyme.

L'application fonctionne sous forme de Single Page Application (SPA) : aucune action ne provoque le rechargement de la page.

Les données sont sauvegardées dans le LocalStorage du navigateur afin d'assurer leur persistance même après fermeture de l'onglet.

---

## Fonctionnalités

### CREATE

Ajout d'une nouvelle idée avec :

* Titre
* Catégorie
* Description

### READ

Affichage dynamique des idées sous forme de cartes.

### UPDATE

Modification du titre et de la description d'une idée existante.

### DELETE

Suppression définitive d'une idée.

### Persistance des données

* Sauvegarde automatique dans le LocalStorage
* Récupération automatique au chargement de la page
